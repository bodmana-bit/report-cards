How to get an API key
  1. Go to console.anthropic.com/settings/keys
  2. Sign up or log in (free account)
  3. Click Create Key, give it a name (e.g. "Report Cards")
  4. Copy the key (starts with sk-ant-) and paste it above
  5. Add a small amount of credit ($5 covers hundreds of report cards)

How to use this tool

A quick guide to getting the best report card comments for your class.

Recommended workflow for 40 students
1
Go to the Spreadsheet tab and download the CSV template. The Qualities reference list on that page shows exactly what to type in the Personality Qualities column.
2
Fill in the spreadsheet in Excel or Google Sheets — one row per student. Save as CSV, open in a text editor, and paste into the Import section.
3
Go to Generate, choose tone and length, and click "Generate all report cards." The AI writes each comment individually.
4
Use the Copy button per student, or Copy all to grab every comment at once. Paste directly into your report card system.
5
Review and personalise — the comments are a strong starting point, but a few of your own touches will make them shine.
Tips for better comments
  • The more specific your strengths and growth notes, the more tailored the comment.
  • Literacy and numeracy notes are woven in naturally — include specific reading levels or strategies if you have them.
  • Select 2–4 personality qualities per student for the best balance.
  • You can add your own custom qualities using the "Add" input — they'll be remembered for that student.
  • If a comment doesn't feel right, click "Regenerate all" to rewrite the full set.
  • BC primary reports focus on the learning journey, not grades. All generated comments are written with this in mind.
Privacy note

Student data is used only to generate comments and is sent to the Anthropic API for processing. It is not stored or used for training. All data lives in your browser tab and is cleared when you close the page. Avoid entering sensitive identifying information beyond what is needed for the comment.
